# Meme Classifier using Deep Learning

## Overview
A deep learning based meme image classification project built using PyTorch.

## Features
- Image classification
- Custom dataset support
- Trained PyTorch model
- API integration

## Tech Stack
- Python
- PyTorch
- OpenCV
- Flask/FastAPI

## Folder Structure

```bash
meme-classifier/
│
├── Data/
├── Model/
├── api/
├── app.py
├── requirements.txt
└── README.md
```
## Installation

```bash
git clone https://github.com/vipin277123-cpu/meme-classifier.git

cd meme-classifier

pip install -r requirements.txt
```
## Usage

```bash
python app.py
```
